An IRC bot that archives IndieWeb event Etherpads to the community wiki.

About the Bot

This archiver is deployed in the IndieWeb chat as cali-iwc.

The IRC bot is active in #indieweb-meta and #indieweb-chat in the IndieWeb chat.

Usage

Archive a Document

To archive a document, use the following syntax:

                
!archive

Example:

                
!archive https://events.indieweb.org/2023/02/homebrew-website-club-europe-london-j3qwd7aBUlxL https://indieweb.org/events/2023-02-15-hwc-london-europe

This will archive the document to the wiki page https://indieweb.org/events/2023-02-15-hwc-london-europe.
